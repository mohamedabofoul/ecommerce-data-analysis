# 🛒 E-Commerce Data Analysis

## 📌 Overview
Analyzed raw e-commerce data to extract meaningful insights and support business decisions using Python and Power BI.

## ⚙️ Process
- Data cleaning and preprocessing using Python  
- Ensured data quality and consistency  
- Built an interactive Power BI dashboard  

## 📊 Dashboard Features
- Total Sales  
- Customer Count & Order Count  
- Average Order Value (AOV)  
- Sales by Month  
- Sales by City  
- Sales by Payment Method  
- Sales Distribution by Age  

## 💡 Business Value
- Identified key sales trends  
- Improved understanding of customer behavior  
- Enabled data-driven decision-making  

## 🛠 Tools Used
- Python (Pandas, NumPy)  
- Power BI
